 # Branding Dersux

 ## Principio general

 Dersux es la plataforma/app.

 Dersu es el concepto de profesional experto y de confianza dentro del relato de marca de Dersux.

 La idea central es:

 - Dersux: la plataforma que conecta y protege.
 - Dersu: la persona profesional que resuelve, cuida y guía.

 ## Jerarquía de nombres

 - Dersux: plataforma/app.
 - Dersu: profesional de confianza dentro de Dersux.
 - Profesional Dersux: término claro para cliente cuando haga falta máxima claridad.
 - Dersu verificado: profesional aprobado/verificado.
 - Dersux Pro: área/cuenta profesional.
 - Los Dersus: uso de comunidad o marketing, con cuidado para no confundir en UI.

 ## Reglas de uso en UI

 - No llamar al profesional solo "Dersux".
 - Evitar frases tipo "un Dersux te ha enviado una solicitud".
 - Usar "Profesional Dersux" cuando haga falta máxima claridad para cliente.
 - Usar "Dersu verificado" cuando se quiera reforzar marca y confianza.
 - Usar "Dersux Pro" para el área profesional.
 - En pantallas legales, admin o técnicas puede mantenerse "profesional" por claridad operativa.

 ## Reglas técnicas

 No se deben cambiar en este proceso de branding:

 - `role = professional`
 - tabla `professionals`
 - `professional_status`
 - rutas `/profesional`
 - nombres de RPCs
 - nombres de SQL
 - nombres de variables/env
 - docs técnicos antiguos donde Arranxos siga siendo contexto histórico

 ## Ejemplos recomendados

 Correctos:

 - "Profesional Dersux"
 - "Dersu verificado"
 - "Conviértete en Dersu"
 - "Dersux Pro"
 - "Encuentra un profesional Dersux"
 - "Tu Dersu ha enviado una propuesta"

 A evitar:

 - "Un Dersux te contactó"
 - "El Dersux aceptó el trabajo"
 - "Dersux disponible"
 - "Cuenta Dersux" si se refiere a profesional; mejor "Dersux Pro"

 ## Copy corto recomendado

 "En Dersux llamamos Dersus a los profesionales verificados: personas expertas que saben resolver, cuidar y guiarte cuando necesitas ayuda en casa."
